# AmplideX® One Reporter

**Release Configuration v2.0 · Software User Guide**

> **Research Use Only.** Not for use in diagnostic procedures.

---

The AmplideX® One Reporter and AmplideX Nanopore Carrier Plus Analysis Module (collectively, "the Software") are a data analysis and visualization tool designed to interpret long-read sequencing FASTQ files generated with Asuragen Carrier Plus kitted products.

| A00630 | AmplideX Nanopore Carrier Plus Kit D (Mix D) |

## Compatible Sequencing Instruments

The Software is compatible with the following Oxford Nanopore Technologies instruments:

- MinION® Mk1B® or Mk1D®
- GridION®
- PromethION®

## MinKNOW Compatibility

The Software analyzes demultiplexed FASTQ outputs generated by MinKNOW®. Each new MinKNOW release is evaluated internally to ensure updates do not affect raw data acquisition or variant detection. Compatibility status is displayed during analysis configuration.

For instructions on analyzing data from older MinKNOW versions, see [Post-run Basecalling](analysis/workflow.md#post-run-basecalling).
